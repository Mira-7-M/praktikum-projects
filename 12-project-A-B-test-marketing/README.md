# praktikum-projects

А/В тест - Маркетинговые мероприятия 

Провела оценку результатов A/B-теста, выявила проблемы

Основные выводы:
- От этапа авторизации до покупки доходит менее 1/3 пользователей.  
- Ожидаемый эффект в части числа пользователей НЕ достигнут.  
- Распределение количества событий на пользователя для групп и распределение выручки с пользователя для групп распределяются похожим образом, однако а группе А их больше.  
- Совпадение с маркетинговыми событиями может сильно повлиять на результаты тестирования, время для него выбрано неудачно.
- Отмечается повышение конверсии только на этапе перехода в корзину, на остальных этапах наблюдается снижение конверсии. 
- необходимо проверить, есть ли проблемы с набором пользователей в тестовую группу или с записью данных о ней.  
- Можно предложить поискать способы повышения конверсии на всех этапах.  

Навыки и инструменты:
- Python, 
- Pandas, 
- NumPy, 
- Matplotlib, 
- seaborn,
- Scipy, 
- Ploty,
- оценка результатов A/B-тестирования,
- проверка гипотез.

# praktikum-projects

Проект по SQL  

Проанализировала базу данных с информацией о книгах, издательствах, авторах, а также пользовательских обзорах книг. 
Эти данные помогают сформулировать ценностное предложение для сервиса по чтению книг.  

Основные выводы:  
- Большая часть книг в базе, 81.9%, вышло после 1 января 2000 года.
- У книг с количеством обзоров менее 10 обзоры и средний рейтинг можно считать довольно предвзятыми.
- Penguin Books - издательство, которое выпустило наибольшее число книг толще 50 страниц.
- Автор с самой высокой средней оценкой книг при учете только книг с 50 и более оценками - J.K. Rowling и Mary GrandPre.
- Среднее количество обзоров от пользователей, которые поставили больше 50 оценок - 24.33

Навыки и инструменты:
- SQL,
- Python, 
- Pandas,  
- Sqlalchemy

# praktikum-projects

Как рассказать историю с помощью данных - Анализ рынка заведений общественного питания Москвы 

Исследовала вопрос - будет ли успешным и популярным в Москве кафе, в котором гостей обслуживают роботы-официанты. 
Проанализирола рынок заведений общественного питания в Москве.

Основные выводы:
- Наиболее популярны в Москве несетевые кафе с количеством посадочных мест около 40.
- Для улиц с большим количеством объектов характерно небольшое количество посадочных мест, так как в самых популярных локациях дорогая стоимость аренды и квадратного метра.
- Для сетевых заведений характерно большое количество заведений с небольшим количеством посадочных мест.
- Среди сетевых объектов общественного питания больше всего доля у предприятий быстрого обслуживания.

Навыки и инструменты:
- Python, 
- Pandas, 
- NumPy, 
- Matplotlib, 
- seaborn,
- Scipy, 
- Random,
- Ploty,
- визуализация данных,
- предобработка данных.

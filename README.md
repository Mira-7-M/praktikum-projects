# praktikum-projects

ПРОЕКТЫ

1.	Предобработка данных - Исследование надёжности заёмщиков банка – Подготовка к построению модели кредитного скоринга

2.	Исследовательский анализ данных  - Исследование объявлений о продаже квартир – Установка параметров для определения рыночной стоимости квартир

3.	Статистический анализ данных  - Анализ трафика.  Определение перспективного тарифа для телеком-компании

4.	Выявление закономерностей, определяющих успешность игры. Проверка гипотез 

5.	Сбор и хранение данных - Аналитика в авиакомпании – Изучение базы данных и анализ спроса пассажиров на рейсы в городах, где проходят крупнейшие фестивали

6.	Анализ бизнес-показателей  - Яндекс.Афиша. Анализ способов снижение бюджетов на маркетинг, отказ  от невыгодных источников трафика и перераспределение бюджета

7.	Принятие решений в бизнесе на основе данных - Гипотезы,  анализ результатов A/B-теста для интернет-магазина, решение о продолжении теста

8.	Как рассказать историю с помощью данных  - Анализ рынка заведений общественного питания Москвы 

9.	Магазин продуктов питания – Исследование поведения пользователей в мобильном приложении.  Исследование результатов A/A/B-эксперимента

10.	Автоматизация  - Дашборд для Яндекс Дзен с анализом пользовательского взаимодействия с карточками статей  – Презентация со ссылкой на Tableau 

11.	Прогнозы и предсказания - Прогнозирование вероятности оттока пользователей для фитнес-центров. Проведение анализа и подготовка плана действий по удержанию клиентов

12.	А/В тест - Маркетинговые мероприятия – Проведение оценки результатов A/B-теста, выявление проблем

13.	Проект по SQL – Анализ базы данных с информацией о книгах, издательствах, авторах, а также пользовательских обзорах книг. Эти данные помогают сформулировать ценностное предложение для сервиса по чтению книг.

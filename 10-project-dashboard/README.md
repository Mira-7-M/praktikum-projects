# praktikum-projects

Автоматизация - Дашборд для Яндекс.Дзен с анализом пользовательского взаимодействия с карточками статей  

Основные выводы:
- Больше всего взаимодействий пользователей с карточками (более 60 тыс.) происходит по теме «Деньги».  
- Наиболее популярные темы источников: «Семейные отношения», «Россия», «Полезные советы», «Путешествия», «Знаменитости» и «Кино».  
- Больше всего соответствий таких тем и карточек источников как «Рассказы» - «Путешествия», «Россия» - «Семейные отношения», «Россия» - «Отношения», «Подборки» - «Полезные советы».
  
Навыки и инструменты:
- Tableau,   
- построение дашбордов,  
- продуктовые метрики.
